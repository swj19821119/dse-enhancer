# DSE Enhancer - 范文和录音生成说明

## 📁 文件说明

### 写作范文
- **文件**: `data/writing-samples.md`
- **内容**: 5道写作题目 × 3等级 = 15篇范文
- **格式**: Markdown，包含题目、范文、评分点分析

### 口语文本稿
- **文件**: `data/speaking-scripts.md`
- **内容**: 5道口语题目 × 3等级 = 15个文本稿
- **格式**: Markdown，包含准备要点和发言稿

### 录音生成脚本
1. **Python脚本**: `scripts/generate-audio.py`
   - 使用Edge-TTS（免费，高质量英音）
   - 需要安装: `pip install edge-tts pydub`
   - 可调节语速、音量

2. **Shell脚本**: `scripts/generate-audio-simple.sh`
   - 使用macOS内置say命令
   - 无需安装额外依赖
   - 适合快速测试

## 🚀 快速开始

### 1. 准备文本文件

将口语文本稿保存为单独的.txt文件：

```
input/
├── speaking-01-level3.txt
├── speaking-01-level4.txt
├── speaking-01-level5.txt
├── speaking-02-level3.txt
└── ...
```

每个文件内容为一篇完整的发言稿。

### 2. 生成录音（Python方案，推荐）

```bash
# 安装依赖
pip install edge-tts pydub

# 运行脚本
python scripts/generate-audio.py
```

**优点**:
- ✅ 完全免费
- ✅ 高质量英音（Microsoft Neural语音）
- ✅ 可调节语速（建议-10%，适合学生）
- ✅ 批量自动处理

### 3. 生成录音（macOS方案，最简单）

```bash
# 赋予执行权限
chmod +x scripts/generate-audio-simple.sh

# 运行脚本
./scripts/generate-audio-simple.sh
```

**优点**:
- ✅ 无需安装
- ✅ 立即可用
- ⚠️ 音质一般，适合测试

## 🎯 范文和录音使用场景

### 学生端
1. **写作练习后**：对比自己的作文与范文，找出差距
2. **口语练习后**：对比自己的录音与示范录音，改进发音和流利度
3. **学习参考**：积累高分词汇和句型

### 教学端
1. **AI评分降级时**：如MVP+3不接入AI，继续使用范文/录音对比方案
2. **辅助评分**：AI评分+范文对比，双重保障
3. **学生自评**：培养自我评估能力

## 💡 内容质量建议

### 写作范文质量分级标准

**Level 3（基础）**:
- 词汇：基础常用词（CEFR A2-B1）
- 句式：简单句为主
- 结构：基本完整
- 字数：80-120词

**Level 4（良好）**:
- 词汇：多样丰富（CEFR B1-B2）
- 句式：长短结合
- 结构：清晰有层次
- 字数：120-180词

**Level 5（优秀）**:
- 词汇：高级学术（CEFR B2-C1）
- 句式：复杂多变
- 结构：严密有深度
- 字数：180-250词

### 口语录音质量分级标准

**Level 3**:
- 语速：较慢，停顿较多
- 词汇：基础日常
- 时长：1.5分钟

**Level 4**:
- 语速：适中，较流畅
- 词汇：学术词汇
- 时长：2分钟

**Level 5**:
- 语速：自然流畅
- 词汇：高级学术
- 时长：2-2.5分钟

## 📝 内容创作Checklist

### 写作范文创作
- [ ] 题目覆盖DSE常考类型（书信、报告、演讲、文章、建议）
- [ ] 每题3个等级差异明显
- [ ] 包含DSE评分三维度（内容、组织、语言）的示范
- [ ] 字数符合DSE要求（Part A: 200-300字）
- [ ] 包含高分词汇和句型批注

### 口语文本创作
- [ ] 题目覆盖DSE常考话题（图片描述、个人经历、观点表达）
- [ ] 提供1分钟准备要点
- [ ] 包含开场白和结束语
- [ ] 时长控制在2分钟左右
- [ ] 词汇难度逐级提升

### 录音制作
- [ ] 使用英音语音
- [ ] 语速适中（Level 3稍慢，Level 5自然）
- [ ] 音质清晰，无背景噪音
- [ ] 文件命名规范（speaking-XX-levelX.mp3）
- [ ] 时长符合等级标准
