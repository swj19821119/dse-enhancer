## 🆕 v2.3 更新（2026-03-25）

### 更新原因
优化无注册体验流程和计分逻辑

### 1. 无注册体验流程调整

**原设计**：
- 访客只能做5题快测 + 每种题型体验3题
- 固定体验题组

**新设计**：
- 访客可完整进行入学自适应测试（15题）
- 测试完成后可进入每日学习（随机题目）
- 学习结果不保存，无法进行自适应学习
- 在无学习进度状态下，可进行每日学习

**注册引导话术**：「注册后可保存学习进度，AI会根据你的弱项智能推送练习，帮你更快提分！」

### 2. 计分逻辑调整：效率得分制

**原设计**：
- 固定题量，按得分计算
- 每日总分 = Σ(模块得分 × 难度权重)

**新设计**：
- 固定时间，不限题量
- 模块得分 = 正确率 × 做题数量 × 难度系数 × 时间效率系数
- 做题越多，正确率越高，得分越高

**激励逻辑**：
- 做题越多，得分上限越高
- 正确率越高，得分越高
- 鼓励在保证正确率的前提下多做题

### 3. 数据库表结构调整

**DailyStudyRecord表新增字段**：
- `total_questions` - 今日总做题数
- `total_correct` - 今日总答对数
- `total_accuracy` - 今日总正确率
- 各模块新增 `*_questions` 和 `*_correct` 字段

### 4. 每日总结报告格式调整

**原格式**：固定满分制（如 词汇：XX/25）

**新格式**：显示做题数量和正确率
```
词汇：XX题 ✓XX 正确率XX% 得分XX
```

## MVP功能范围（调整后）

### ✅ 保留功能（8个核心功能）

1. **用户系统**
   - 注册/登录（邮箱+密码）
   - 无注册体验（访客模式）
   - 个人档案（年级、目标等级）
   - VIP状态（基础字段）

2. **入学自适应测试**
   - 年级选择（小学到中六）
   - 15题自适应测试
   - 能力定级（6维度）
   - 能力图谱展示

3. **每日学习流程**
   - 轻松学模式（35分钟）
   - 勤奋学模式（45分钟）
   - 4个模块：词汇、语法、阅读、错题复习
   - 进度追踪和完成报告

4. **词汇模块**
   - 单词卡片展示
   - Part A/Part B区分
   - 生词本（认识/不认识）
   - **浏览器TTS发音**（Web Speech API）✨

5. **语法模块**
   - 按难度分级（1-5级）
   - 按知识点分类
   - 即时反馈和解析

6. **阅读模块**
   - 文章阅读
   - 生词点击查词
   - 选择题答题

7. **错题本**
   - 错题记录
   - 错题复习
   - 间隔重复（简化版）

8. **数据看板**
   - 首页统计
   - 学习记录
   - 能力图谱

### ❌ 移除功能（MVP后开发）

1. **PK对战** - 需要Redis + WebSocket
2. **排行榜** - 依赖PK对战
3. **AI写作评分** - 需要ArkCode/GPT-4o API
4. **AI口语评分** - 需要Whisper ASR
5. **听力模块** - 需要TTS音频生成
6. **亲子账号** - 复杂度较高
7. **完整真题模考** - 需要大量题目数据

## 风险与应对

### 风险1：Web Speech API兼容性
- **风险**：部分浏览器不支持
- **应对**：提供降级方案（显示音标）
- **代码**:
```typescript

}
```

### 风险2：SQLite并发性能
- **风险**：并发写入性能差
- **应对**：MVP阶段用户量少，后续迁移到PostgreSQL

### 风险3：无AI评分体验下降
- **风险**：写作/口语无AI评分
- **应对**：
  - 提供详细范文对比
