# DSE Enhancer - Product Requirements Document

**项目名称**: DSE Enhancer
**定位**: 香港DSE英语AI自适应学习平台，专为在港大陆学生设计
**核心 slogan**: 每天40分钟，精准提高DSE英语成绩
**MVP阶段**: 2026-Q1

---

## 1. 产品定位

### 目标用户
- 在港就读的大陆学生
- DSE英语成绩普遍在2-3级，目标提升到3-5级
- 每天放学后可以抽出40分钟集中学习

### 核心痛点
1. 大陆学生基础不差，但不适应DSE的题型和出题思路
2. 香港本地培训班太贵，时间不灵活
3. 现有线上平台内容太泛，不针对DSE垂直场景
4. 缺乏个性化自适应，不知道每天该练什么

### 差异化
- ✅ 垂直只做DSE英语，完全贴合考纲
- ✅ AI自适应学习，每天自动安排针对性练习
- ✅ 一键开始，40分钟完成，碎片化设计贴合课后场景
- ✅ AI评分写作口语，即时反馈，不需要等老师批改
- ✅ 游戏化PK排行榜，激励坚持学习

---

## 2. 核心业务流程

### 整体流程

```
用户访问 → 注册/登录 → 选择年级 → 入学自适应测试 → 能力定级 → 生成初始学习计划
    ↓
每日访问 → 点击【一键开始今日40分钟】→ 系统自动组合内容 → 按顺序完成四个模块
    ↓
每日结算 → 各模块计分 → 生成总分 → 分析弱项 → 调整明日练习 → 打卡完成
    ↓
持续迭代 → 能力提升 → 调整难度 → 直到达到目标分数
```

### 关键设计：一键开始每日学习
用户不需要自己选内容，系统根据：
1. 当前能力水平
2. 历史得分弱项
3. 错题知识点
4. 学习进度

自动组合成约40分钟，具体轮换安排：

| 模块 | 用时 | 频率 | 具体内容 |
|------|------|------|----------|
| 词汇 | 10分钟 | ✅ 每天都有 | 闯关背单词/生词本复习 |
| 语法 | 10分钟 | ✅ 每天都有 | 语法专项练习 |
| 阅读/听力/写作(Part A)/口语(Part A) | 10分钟 | **交替轮换** | 见下方轮换表 |
| 错题复习 | 10分钟 | ✅ 每天都有 | 错题间隔重复复习 |

**轮换安排（每周循环）**：
- 第1天：阅读
- 第2天：听力
- 第3天：写作 Part A（情景写作，短篇200-300字，限时10分钟）
- 第4天：阅读
- 第5天：听力
- 第6天：口语 Part A（个人独白，1分钟准备+2分钟发言，限时10分钟）
- 第7天：阅读/听力弱项优先

总共约40分钟，符合用户场景。

---

### 写作和口语 Part 设计说明

| 模块 | 日常打卡（一键学习） | 专项练习（自由练习） |
|------|-------------------|-------------------|
| **写作** | 只练 Part A（情景写作，200-300字，限时10分钟） | Part A + Part B（完整写作练习，无时间限制） |
| **口语** | 只练 Part A（个人独白，1分钟准备+2分钟发言，限时10分钟） | Part A + Part B（完整口语练习，无时间限制） |

日常打卡只练单个 Part，控制在10分钟内；专项练习可以练完整 Part A + Part B。

## 3. 功能模块详情

### 3.1 用户体系

#### 基础账号
| 功能 | 说明 |
|------|------|
| 注册登录 | 手机号 + 验证码登录（支持微信/Google快捷登录） |
| 个人档案 | 年级、目标分数、当前等级、学习天数、累计积分 |
| 能力图谱 | 可视化展示听/说/读/写四个维度的当前等级 |
| 会员系统 | 免费试用7天，付费会员解锁全部功能 |

#### 亲子账号体系（解决家长帮孩子注册问题）

**场景**：很多是家长付费，孩子使用，家长需要看进度

**方案设计**：
- 主账号：可以是家长注册，也可以是孩子自己注册
- 主账号可以**添加多个子账号**（对应多个孩子）
- 每个孩子一个独立子账号，有独立的学习进度、错题本、能力图谱
- 孩子登录使用子账号，做题练习
- 家长登录主账号，可以：
  1. 查看每个孩子的学习报告
  2. 管理会员订阅（一个会员可以支持多个孩子？或按孩子数收费）
  3. 接收每周学习汇总邮件

**账号关系图**：
```
主账号（家长/学生本人）
  ├─ 子账号 老大 → 独立学习数据
  ├─ 子账号 老二 → 独立学习数据
  └─ 子账号 老三 → 独立学习数据
```

**登录流程**：
- 家长扫码/输入验证码登录主账号
- 在主账号仪表盘选择要查看哪个孩子，或者切换到孩子账号学习
- 孩子也可以直接用子账号手机号登录（如果有自己手机）
- 支持快捷切换账号，不需要重新登录

**付费方案**：
- 基础会员：支持 1 个孩子
- 家庭会员：支持最多 3 个孩子（优惠价）
- 这样满足多孩家庭需求，也增加ARPU

### 3.7 语法专项练习

**业务流程**：
- 按DSE常考语法点分类（时态/冠词/从句/虚拟语气/非谓语动词...）
- 大陆学生常见弱项重点标注（比如时态、冠词、虚拟语气是大陆学生普遍弱项）
- 按难度分级：1-5级对应不同年级水平
  - 中三：基础语法（时态、冠词、简单从句）
  - 中四：中级语法（状语从句、定语从句、非谓语动词）
  - 中五/中六：高级语法（虚拟语气、倒装、强调句型）
- 每次10题，做完马上改分，看详细解析
- 错题自动收藏

### 3.8 写作练习模块

DSE英语写作分为 **Paper 2 Writing**，包含两部分：
- **Part A**：情景写作（必答，建议作答时间约40分钟，字数约200-300字）
  - 常见题型：书信、报告、便条、演讲辞、文章投稿、社论
- **Part B**：议题写作（三选一，建议作答时间约60分钟，字数约400-600字）
  - 常见题型：议论文、说明文、记叙文、散文

#### 移动端优化：语音输入作文（特色功能）

为方便移动端用户，支持 **语音直接输入作文**：

**业务流程**：
1. 用户可以选择「键盘输入」或「语音输入」
2. 选择「语音输入」→ 按住录音按钮开始说话
3. Whisper ASR 实时转文字，显示在输入框中
4. 用户可以编辑修改转写的文字
5. 提交后和普通输入一样进行AI评分

**优势**：
- ✅ 移动端用户更方便，不用敲键盘
- ✅ 同时锻炼口语表达能力
- ✅ 提升写作速度，降低门槛
- ✅ 支持普通话→英文？或者英文→英文？（默认英文语音输入）

**业务流程**：

```
进入写作 → 选择考试部分（Part A/Part B）→ 选择题目（真题/模拟）
    ↓
    ┌──────────────────────────────────┐
    │  题目要求展示（情景/议题）       │
    │  字数要求：XX-XX                │
    │  建议时间：XX分钟               │
    │  题目原文：...                 │
    └──────────────────────────────────┘
    ↓
    用户输入作文 → 提交AI评分
    ↓
    AI按照DSE官方评分标准从三个维度评分：
    1. 内容 (Content)：扣题性、完整性、观点清晰度
    2. 组织 (Organization)：结构连贯、段落划分、逻辑衔接
    3. 语言 (Language)：词汇丰富度、语法准确性、拼写标点
    ↓
    AI返回详细评分报告：
    ┌──────────────────────────────────┐
    │  总分：XX/5       DSE等级：Level X │
    │                              │
    │  内容：X/5     ⭐⭐⭐⭐           │
    │  组织：X/5     ⭐⭐⭐            │
    │  语言：X/5     ⭐⭐⭐⭐           │
    │                              │
    │  ✍️ 优点：                     │
    │  - 观点清晰，紧扣主题          │
    │                              │
    │  🔍 需要改进：                 │
    │  - 第2段缺乏逻辑连接词         │
    │  - 这些词汇可以升级更高级表达   │
    │  - 此处语法错误：...          │
    │                              │
    │  [查看参考范文]               │
    │  [全文纠错]                   │
    └──────────────────────────────────┘
```

**功能特性**：
- 完全按照DSE Part A/Part B题型分类训练
- 支持字数统计，提醒是否符合考试要求
- 逐句语法拼写检查，标记错误并给出修正
- 词汇升级建议，帮助学生从基础词汇升级到DSE高分词汇
- 保存写作历史，可以对比多次练习进步

### 3.9 口语练习模块

DSE英语口语考试为 **Paper 4 Speaking**，分为两部分：
- **Part A**：个人独白（约3分钟，1分钟准备，2分钟发言）
  - 考官给出一张图片和一个话题，考生根据图片和话题做个人阐述
- **Part B**：小组讨论（约7分钟，1分钟准备，5-6分钟讨论）
  - 给定一个议题，小组成员（通常4人）进行讨论互动

**业务流程（Part A 个人独白）**：

```
进入口语 → 选择Part A → 选择题目（真题/模拟）
    ↓
    展示题目图片 + 话题 → 开始1分钟倒计时准备
    ↓
    倒计时结束 → 点击开始录音
    ↓
    录音完成（约2分钟）→ 提交
    ↓
    AI处理：
    1. Whisper ASR转文字（英音准确识别）
    2. 发音评分：
       - 单词发音准确性
       - 重音和语调
       - 流利度（停顿频率、流畅性）
    3. 内容评分：
       - 扣题性
       - 观点完整性
       - 词汇语法准确性
    ↓
    输出评分报告：
    ┌──────────────────────────────┐
    │  总分：XX/5                   │
    │  发音：X/5   ⭐⭐⭐⭐           │
    │  内容：X/5   ⭐⭐⭐            │
    │                              │
    │  发音不准单词标红：            │
    │  - "environment" 重音错       │
    │  流利度：较好，仅有2处停顿    │
    │                              │
    │  内容改进建议：                │
    │  - 可以多增加一个例子支持观点  │
    └──────────────────────────────┘
```

**业务流程（Part B 小组讨论）**：

```
进入口语 → 选择Part B → 选择题目（真题/模拟）
    ↓
    展示议题题目 → 1分钟准备
    ↓
    开始录音 → 用户进行讨论发言（模拟对话题）
    ↓
    提交后AI评分：
    - 互动性：是否回应他人观点、能否引导讨论
    - 观点清晰度：表达是否清晰
    - 语言准确性：词汇语法
    - 发音流利度
    ↓
    输出详细报告，给出互动建议
```

**功能特性**：
- 分Part A/Part B独立训练，完全贴合考试
- Part A包含图片题库，还原真实考试场景
- 逐词发音分析，标记不准单词并给出示范读音
- 流利度分析，提醒卡壳停顿位置
- 支持反复练习，对比进步

### 3.13 自适应调整算法

```
每日总分 = Σ(模块得分 × 难度权重)

调整规则：
1. 某模块得分 < 60 → 明日增加该模块练习权重
2. 某模块得分 > 85 → 明日升高难度，减少基础练习
3. 错题知识点 → 自动加入第二天错题复习
4. 保持每日总时长约40分钟
```

## 4. 题库建设策略

### MVP阶段
1. **真题整理**：整理公开的DSE历年真题（2012-2024），用于产品验证
2. **AI辅助出题**：用大模型按照DSE题型和难度生成新题，人工抽检
3. **听力音源**：TTS生成，成本极低
4. **作文口语**：AI评分，不需要人工

### 后续扩容
1. 每年更新最新真题
2. 用户报错反馈，人工审核更新题库
3. 逐步换成真人录音听力

### 标签体系
每道题必须打标签：
- 题型：词汇/语法/阅读/听力/写作/口语
- 知识点：具体语法点/主题
- 难度：1-5级（对应DSE 1-5级水平）
- 目标年级：form3/form4/form5/form6（中三到中六）
- DSE年份：如果是真题

### 难度分级与年级对应

我们支持从小学三年级到中六的全学段用户：

| 难度等级 | 对应DSE水平 | 对应年级 | 说明 |
|---------|------------|---------|------|
| 预备级 | 基础启蒙 | 小三-小六 | 剑桥五级KET/PET水平，DSE前置基础 |
| 1级 | Level 1-2 | 中三基础 | 核心基础词汇，最简单语法点 |
| 2级 | Level 2-3 | 中三/中四 | 高频词汇，基础语法，短篇阅读 |
| 3级 | Level 3-4 | 中四/中五 | 中频词汇，复杂语法，中篇阅读 |
| 4级 | Level 4-5 | 中五/中六 | 低频高阶词汇，复杂句型，长篇阅读 |
| 5级 | Level 5 以上 | 中六冲刺 | 核心考纲全覆盖，真题难度 |

**自适应推送逻辑**：
- 用户注册时选择年级（支持小三到中六）→ 初始难度匹配该年级平均水平
- 入学测试后根据实际水平调整难度（不一定完全匹配年级）
- 每次练习后根据得分动态调整，不会给学生推送远超出当前水平的题目
- 例如：小学五年级学生如果基础好，可以跳到预备级以上；中三学生如果测试达到3级水平，会推送2-3级题目

### 词汇分级策略

我们支持从小学到DSE全程词汇学习，总词汇量覆盖：

| 级别 | 词汇量 | 对应难度 | 对应年级 | 说明 |
|------|--------|----------|---------|------|
| 预备级 | ~1200 | 预备级 | 小三-小六 | 小学进阶词汇，KET/PET基础，为DSE铺垫 |
| 一级 | ~1300 | 1-2级 | 中三基础 | DSE最高频考词，核心基础 |
| 二级 | ~1500 | 2-3级 | 中三/中四 | DSE次高频考词 |
| 三级 | ~1500 | 3-4级 | 中四/中五 | DSE常考中频词 |
| 四级 | ~1000 | 4-5级 | 中五/中六 | DSE低频高阶词 |
| 五级 | ~500 | 5级 | 中六冲刺 | DSE最难核心考词 |

**总词汇量**：约 7000词，完全覆盖DSE要求 + 小学预备阶段

**按年级词汇学习进度建议**：
- **小学三年级**：开始预备级，逐步积累
- **小学六年级毕业**：完成预备级（~1200词），具备DSE入门基础
- **中三（Form 3）**：完成一级 + 二级词汇，累计 ~4000词
- **中四（Form 4）**：完成三级词汇，累计 ~5500词
- **中五（Form 5）**：完成四级词汇，累计 ~6500词
- **中六（Form 6）**：掌握五级词汇，冲刺总词汇 ~7000词

**词汇排序原则**：
- 严格按照DSE过去10年真题考频排序，先背考频高的词
- 同一词汇只会出现在一个难度级别，不重复
- 按主题分组（教育、科技、环境、文化、社会...）方便针对性练习

## 1. 技术选型

### 整体架构
- **前端**：React 18 + TypeScript + Tailwind CSS + Shadcn UI
  - 原因：组件丰富，开发快，样式美观，适合MVP
  - Tailwind Utility-first 快速开发

- **后端**：Node.js + Express / Next.js Full Stack
  - 原因：全JS栈，开发效率高，前后端可以一人维护
  - 选择Next.js = 前端+后端一体化，部署简单

- **数据库**：PostgreSQL
  - 原因：支持复杂查询，结构化数据适合本题库系统
  - 腾讯云PostgreSQL托管，省心

- **缓存**：Redis (可选，MVP可以不用，后期加)

- **AI能力**：
  - 作文/口语内容评分：调用ArkCode / GPT-4o API
  - ASR语音识别：OpenAI Whisper (本地部署或API)
  - TTS语音合成：Google Text-to-Speech / 阿里云TTS API
  - 发音评分：可以用Whisper+比对，或第三方API

- **部署**：
  - 服务器：腾讯云Ubuntu，已经购买
  - Docker + Docker Compose 容器化部署
  - Nginx 反向代理 + SSL证书 (Let's Encrypt)
  - CI/CD：GitHub Actions → 代码push自动构建部署

1. **TypeScript**：所有代码必须加类型，禁止 `any`
2. **命名**：驼峰命名，函数名动词开头，如 `getUser()`
3. **组件**：一个文件一个组件，组件名大驼峰
4. **API**：统一返回格式 `{code: number, data: T, message: string}`
5. **错误处理**：所有异步操作必须捕获错误
6. **Git**：每次提交一个清晰的功能，commit message 语义化

## 12. 竞品分析与差异化补充

对比现有DSE学习平台（例如Snapask、Khan Academy、香港本地补习班APP），我们还需要补充这些用户需求：

### 12.1 大陆学生针对性弱项强化

**未满足需求**：
- 大陆学生的DSE英语弱项和香港本地学生不一样
- 例如：冠词使用、时态一致性、英式拼写、议论文结构，都是大陆学生高频错点

**方案**：
- 建立「大陆学生高频错题库」，专门收集统计大陆学生常错的语法点
- 算法自动强化练习这些弱项，比通用平台更有针对性
- 每个语法点都配有「大陆学生常见错误提示」

### 12.2 繁简体切换

**未满足需求**：
- 大陆学生习惯看简体字，现有平台都是繁体，阅读不习惯

**方案**：
- 全站支持繁简体一键切换
- 默认根据用户设备语言自动选择
- 词汇解释、题目说明都对应转换

### 12.3 生词本导出打印

**未满足需求**：
- 很多学生还是喜欢纸质背单词，需要导出打印

**方案**：
- 支持生词本导出为PDF格式，可打印
- 多种排版模板：乱序/按顺序、带音标/不带音标
- 会员功能，增加付费价值

### 12.4 真题套卷模考

**未满足需求**：
- 到了中六冲刺阶段，学生需要完整模考适应考试节奏
- 现有MVP范围只覆盖每日练习，没做完整模考

**方案**：
- 放在后续阶段开发，不影响MVP上线
- 完整模考包括：Paper 1阅读+Paper 2写作+Paper 3听力+Paper 4口语
- 自动计时，考完出总分预测DSE等级

### 12.5 学习计划制定

**未满足需求**：
- 不同目标分数（比如从3到5需要多长时间），学生不知道每天该练多少

**方案**：
- 注册时输入「当前等级」→「目标等级」→「考试时间」
- 系统自动计算每天需要完成的练习量
- 进度条跟踪，提醒每天是否达标

### 12.6 家长端监控

**未满足需求**：
- 很多是家长付费，家长想知道孩子学习进度

**方案**：
- 支持绑定家长账号，每周发送学习报告邮件
- 报告内容：学习天数、完成练习量、弱项进展
- 不侵犯孩子隐私，只展示汇总数据，不看具体做题内容

### 12.7 离线练习

**未满足需求**：
- 学生可能在通勤路上学习，没网络

**方案**：
- APP端支持提前下载题目，离线练习
- 上线后自动同步进度
- MVP先做Web端，后期做APP再加

### 12.8 错题导出

**未满足需求**：
- 学生考前复习需要把所有错题集中再过一遍

**方案**：
- 支持按题型/知识点导出错题
- 可以生成PDF打印，或导出CSV
- 会员功能

### 12.9 发音对比

**未满足需求**：
- 学生不知道自己发音错在哪里，只知道得分不够

**方案**：
- 发音评分后，给出标准读音的示范录音
- 支持学生播放自己的录音和标准录音对比
- 波形图对比，直观看到语速和重音差异

### 12.10 考试倒计时

**未满足需求**：
- 中六学生需要时刻提醒距离DSE还有多少天

**方案**：
- 首页显示考试倒计时
- 根据剩余天数调整每日练习量建议

## 13. MVP优先级调整建议

| 功能 | 原计划 | 建议调整 | 原因 |
|------|--------|----------|------|
| 繁简体切换 | 无 | MVP就做 | 大陆用户核心体验，必须有 |
