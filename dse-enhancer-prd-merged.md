## 1. 产品定位

### 目标用户
- 在港就读的大陆学生
- DSE英语成绩普遍在2-3级，目标提升到3-5级
- 每天放学后可以抽出40分钟集中学习

### 核心痛点
1. 大陆学生基础不差，但不适应DSE的题型和出题思路
2. 香港本地培训班太贵，时间不灵活
3. 现有线上平台内容太泛，不针对DSE垂直场景
4. 缺乏个性化自适应，不知道每天该练什么

### 差异化
- ✅ 垂直只做DSE英语，完全贴合考纲
- ✅ AI自适应学习，每天自动安排针对性练习
- ✅ 一键开始，40分钟完成，碎片化设计贴合课后场景
- ✅ 范文写作自评，即时反馈，不需要等老师批改
- ✅ 大陆学生针对性弱项强化，完全不同于香港本地平台
- ✅ 繁简体切换，阅读体验友好
- ✅ 支持香港学制（P1-P6, F1-F6）

## 2. 核心业务流程

### 2.1 整体流程

```
用户访问 → 选择年级（P1-P6, F1-F6）→ 入学测试（15题）→ 能力定级显示
    ↓
注册/登录 → 选择年级（P1-P6, F1-F6）→ 入学测试（15题）→ 能力定级
    ↓
每日访问 → 点击【一键开始今日学习】→ 系统自动组合内容 → 按顺序完成四个模块
    ↓
每日结算 → 各模块计分 → 生成总分（满分100）→ 分析弱项 → 调整明日练习
```

### 2.2 年级选择（香港学制）

| 年级范围 | 对应学制 | 能力等级范围 |
|---------|---------|-------------|
| P1-P6 | 小学一至六年级 | Pre Level（预备级，不计入DSE 1-5级）|
| F1-F3 | 中一至中三 | Level 1-2（基础级）|
| F4-F6 | 中四至中六 | Level 2-5（DSE备考级）|

**注意**：P1-P6小学生使用Pre Level难度体系，为将来DSE学习打基础；F1-F6学生按DSE标准评定Level 1-5。

### 2.3 两种学习模式

#### 模式 1：轻松学（40分钟）

| 模块 | 用时 | 题量 |
|------|------|------|
| 词汇 | 10分钟 | 15-20题 |
| 语法 | 10分钟 | 10-15题 |
| 阅读 | 10分钟 | 2-3篇 |
| 听力 | 10分钟 | 1-2篇 |
| **总计** | **40分钟** | **约40题** |

#### 模式 2：勤奋学（60分钟）

| 模块 | 用时 | 说明 |
|------|------|------|
| 轻松学核心模块 | 40分钟 | 词汇 + 语法 + 阅读 + 听力 |
| 错题复习 | 5分钟 | 艾宾浩斯间隔重复复习 |
| 综合练习 | 15分钟 | 词汇/听力Part B轮换 |
| **总计** | **60分钟** | 深度学习模式 |

**勤奋学每周轮换安排**：

| 星期 | Part B 补充 |
|------|-------------|
| 周一 | 听力 Part B |
| 周二 | 词汇 Part B |
| 周三 | 语法 Part B |
| 周四 | 阅读 Part B |
| 周五 | 写作 Part A |
| 周六 | 口语 Part A |
| 周日 | 休息日（不强制一键学习）|

## 3. 功能模块详情

### 3.1 用户体系

#### 注册登录
- 邮箱 + 密码登录
- 个人档案：年级、目标等级、学习天数

#### 能力图谱
可视化展示听/说/读/写四个维度的当前等级

### 3.2 入学测试

**测试目标**：根据学生年级和答题表现，评定当前能力等级，建立个性化学习起点。

#### 年级与能力等级对应

| 选择年级 | 学制 | 能力等级范围 | 起始难度 |
|---------|------|-------------|----------|
| P1-P3 | 小学低年级 | Pre Level (0.5-0.9) | 1.0 |
| P4-P6 | 小学高年级 | Pre Level (0.7-1.0) | 1.5 |
| F1-F2 | 中一至中二 | Level 1 (1.0-1.9) | 2.0 |
| F3 | 中三 | Level 2 (2.0-2.9) | 2.5 |
| F4 | 中四 | Level 3 (3.0-3.9) | 3.0 |
| F5 | 中五 | Level 4 (4.0-4.9) | 3.5 |
| F6 | 中六 | Level 5 (5.0-5.9) | 4.0 |

#### 测试流程

```
1. 用户选择年级（P1-P6, F1-F6）
2. 15题自适应测试（约10分钟）
   - 起始难度根据年级设定
   - 答对 → 下一题难度+0.1
   - 答错 → 下一题难度-0.1
   - 难度范围: 1.0-5.0
3. 计算最终等级（1位小数）
4. 输出能力报告
```

#### 能力等级体系（基于DSE官方标准）

| 等级 | CEFR对应 | 能力描述 | 适用年级 |
|------|----------|----------|----------|
| Pre Level (0.5-1.0) | Pre-A1 ~ A1 | 基础入门：掌握基础词汇和简单句型 | P1-P6 |
| Level 1 (1.0-1.9) | A2 | 初级水平：理解常用表达，能在熟悉情境下进行简单沟通 | F1-F2 |
| Level 2 (2.0-2.9) | B1 | 中级水平：理解一般性话题，能应对日常情境 | F2-F3 |
| Level 3 (3.0-3.9) | B2 | 良好水平：理解复杂文本，能流利表达观点 | F3-F4 |
| Level 4 (4.0-4.9) | C1 | 优秀水平：理解高难度文本，能灵活准确使用语言 | F5-F6 |
| Level 5 (5.0-5.9) | C1+ ~ C2 | 卓越水平：接近母语者水平 | F6冲刺 |

### 3.4 学习模块

#### 词汇模块
- Part A：高频核心词（日常练习，Level 1-3难度）
- Part B：低频高阶词（专项练习，Level 4-5难度）

#### 语法模块
- Part A：基础语法（时态、冠词、从句）
- Part B：高级语法（虚拟语气、倒装、强调句型）

#### 阅读模块
- 短文理解，选择+简答
- 点击生词查看翻译

#### 听力模块
- Part A：选择题/填空题（短对话/独白）
- Part B：综合任务（较长录音+阅读材料）
- 使用TTS生成音频（Web Speech API）

### 3.5 写作练习模块

**MVP低成本方案**：范文对比 + 自评清单

```
进入写作 → 选择题目 → 用户输入作文
    ↓
提交后显示：
- 字数统计
- 自评清单（内容/组织/语言）
- 参考范文（Level 3/4/5各一篇）
```

### 3.6 口语练习模块

**MVP低成本方案**：录音对比 + 自评清单

```
进入口语 → Part A个人独白（1分钟准备+2分钟发言）
    ↓
录音保存，显示：
- 录音回放
- 自评清单（发音/流利度/内容/互动）
- 参考录音（Level 3/4/5各一份）
```

### 3.7 错题本模块

基于艾宾浩斯遗忘曲线，设置8个复习阶段：

| 阶段 | 间隔时间 |
|------|---------|
| 第1次复习 | 5分钟后 |
| 第2次复习 | 30分钟后 |
| 第3次复习 | 12小时后 |
| 第4次复习 | 1天后 |
| 第5次复习 | 2天后 |
| 第6次复习 | 4天后 |
| 第7次复习 | 7天后 |
| 第8次复习 | 15天后（掌握）|

**复习规则**：
- 做对 → stage + 1（最高8，完全掌握）
- 做错 → stage - 2（最低1）

### 3.8 计分逻辑

#### 边际递减得分制

**基础得分**（满分60）：
```
基础得分 = 60 × (正确率 / 0.6) × (难度系数 / 3)
```
- 正确率60%为基准线
- 难度系数：Level 1=1.0, Level 5=1.5

**额外加分**（满分40，边际递减）：
```
额外加分 = 额外做题数 × 正确率系数 × (1 / (1 + 0.1 × 额外做题数))
```
- 每多做1题，边际效益递减10%

**示例**：
- 默认15题，完成25题（额外10题），正确率85%
- 额外加分 = 10 × 1.0 × (1 / (1 + 0.1 × 10)) = 5分
- 模块总分 = 60 + 5 = 65分

#### 自适应难度调整

| 正确率 | 难度调整 |
|--------|---------|
| < 60% | 难度 -0.1 |
| 60-80% | 保持 |
| > 80% | 难度 +0.05 ~ +0.1 |

## 9. API接口设计

### 统一响应格式

**成功响应**：
```json
{
  "success": true,
  "data": { ... },
  "message": "操作成功"
}
```

**注意**：认证相关API（登录/注册）返回 `user` 和 `token` 在顶层：
```json
{
  "success": true,
  "user": { "id": "uuid", "email": "...", "nickname": "张三" },
  "token": "jwt-token",
  "message": "登录成功"
}
```

**错误响应**：
```json
{
  "success": false,
  "error": "错误描述"
}
```

### 认证相关

#### POST /api/auth/register/ - 注册
```json
// 请求
{ "email": "user@example.com", "password": "123456", "nickname": "张三" }

// 成功响应
{
  "success": true,
  "user": { "id": "uuid", "email": "...", "nickname": "张三" },
  "token": "jwt-token"
}
```

#### POST /api/auth/login/ - 登录
```json
// 请求
{ "email": "user@example.com", "password": "123456" }

// 成功响应
{
  "success": true,
  "user": { "id": "uuid", "email": "...", "nickname": "张三" },
  "token": "jwt-token"
}
```

### 入学测试

#### POST /api/placement/start/ - 开始测试
```json
// 请求
{ "grade": "f4" }

// 响应
{
  "success": true,
  "data": {
    "sessionId": "uuid",
    "questions": [...],
    "totalQuestions": 15
  }
}
```

#### POST /api/placement/submit/ - 提交答案
```json
// 请求
{ "sessionId": "uuid", "answers": [{ "questionId": "q1", "answer": "A" }] }

// 响应
{
  "success": true,
  "data": {
    "level": 2.3,
    "correctCount": 10,
    "totalCount": 15,
    "accuracy": 66.7
  }

```

### 每日学习

#### POST /api/study/session/start/ - 开始学习
```json
// 请求
{ "mode": "relaxed" }

// 响应
{
  "success": true,
  "data": {
    "sessionId": "uuid",

    "firstModule": "vocabulary",
```

